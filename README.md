# NSU-Cyber-Lecture-Downloader
Namseoul Univ. (NSU) Cyber Lecture Downloader (https://lms.nsu.ac.kr)  
위 프로그램을 사용하기 위해선 ffmpeg를 path에 등록해두거나 main.py 파일과 같은 경로에 ffmpeg.exe 파일을 위치시켜야 합니다.
